SetDrawColor

SetDrawColor(int r [, int g, int b])

Leírás

Beállítja a színt, amely az összes rajzolási művelethez használatos (vonalak, téglalapok, cella keretek). Kifejezhető RGB komponensekkel vagy szürkeárnyalatosan. A függvény meghívható az első oldal készítése előtt és értéke megmarad oldalról oldalra.

Paraméterek

r
Ha g és b megadott, piros komponens; Ha nem, szürkeárnyalat. Értéke 0 - 255 között.
g
Zöld komponens (érték 0 - 255 között).
b
Kék komponens (érték 0 - 255 között).

Lásd még

SetFillColor, SetTextColor, Line, Rect, Cell, MultiCell
