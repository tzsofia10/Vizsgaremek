Output

string Output([string dest [, string name [, boolean isUTF8]]])

Leírás

Elküldi a dokumentumot a megadottak szerinti végcélba: böngészőbe, fájlba vagy srtingbe. A böngészőbe való küldés esetén a beépülő modul kerül használatra vagy a letöltés ("Mentés másként" ablak) lesz kikényszerítve.
A függvény először a Close() -t hívja meg és ha szükséges lezárja a dokumentumot.

Paraméterek

dest
Végcél ahová a dokumentum elküldésre kerül. Ezen értékek egyikét veheti fel:
  • I: egyenesen a böngészőbe küldi a dokumentumot. Beépülő modul lesz használva, amennyiben elérhető.
  • D: elküldi a dokumentumot a böngészőnek és kikényszeríti a letöltést. A letöltött fájlnevet a name adja meg.
  • F: elmenti a dokumentumot egy helyi fájlba a name névvel (tartalmazhat még elérési utat is a fájlnéven kívül).
  • S: a dokumentumot egy stingben adja vissza.
Alapértéke I.
name
A fájl neve. Átugrásra kerül, amennyiben a célállomás S.
Alapértéke doc.pdf.
isUTF8
Beállítható, ha a név ISO-8859-1 (false) vagy UTF-8 (true) enkódolású. Kizárólag I és D célállomás esetén használható.
Alapértéke false.
