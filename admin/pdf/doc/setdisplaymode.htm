SetDisplayMode

SetDisplayMode(mixed zoom [, string layout])

Leírás

Beállítja a módot, hogy a dokumentum hogyan jelenjen meg a képernyőn. A nagyítási szint beállítható: egész oldal; oldalszélességben; eredeti méretet használva; nagyítási szintet megadva; használva a program alapértelmezését (beállítható az Adobe Reader Preferences menüjében). Az oldalak elrendezése is megadható: egyszerre csak egy; folyatólagos; két oszlopban; a program alapértelmezéseként.

Paraméterek

zoom
A nagyítás mértéke. Az alábbi értékek valamelyikét veheti fel:
  • fullpage: az egész oldalt megjeleníti
  • fullwidth: oldalszélesség
  • real: eredeti méret (egyenlő a 100%-os nagyítással)
  • default: a program alapértelmezett értéke
vagy a nagyítás mértékét megadó számmal is megadható.
layout
Az oldalak elrendezése. Lehetséges értékek:
  • single: egy oldalt jelenít meg
  • continuous: folytatólagosan jelennek meg az oldalak
  • two: két oldal két oszlopban jelenik meg
  • default: a program alapértelmezése
Alapértelmezés: default.
