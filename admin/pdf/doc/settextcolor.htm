SetTextColor

SetTextColor(int r [, int g, int b])

Leírás

A betűszínt állítja be. Kifejezhető RGB komponensekkel vagy szürkeárnyalatosan. A függvény meghívható az első oldal készítése előtt így az értéke lapról lapra továbbadódik.

Paraméterek

r
Ha g és b megadott akkor piros komponenst jelöl; Ha nem, a szürkeárnyalat mértékét adja meg. Értéke 0 és 255 között.
g
Zöld komponens (értéke 0 és 255 között).
b
Kék komponens (értéke 0 és 255 között).
