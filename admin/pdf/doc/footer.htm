Footer

Footer()

Leírás

Ez a függvény felel az oldal láblécének beillesztéséért. Automatikusan meghívásra kerül az AddPage() és a Close() által, így nem érdemes közvetlenül meghívni. Alapértelmezettként az FPDF-ben üres, így be kell állítani, ha láblécet szeretnénk.

Példa

class PDF extends FPDF
{
function Footer()
{
    // Ugrás 1,5 cm-rel alulról
    $this->SetY(-15);
    // Arial, dőlt, 8 pt-os betűtípus kiválasztása
    $this->SetFont('Arial','I',8);
    // Középre igazított oldalszám kiírása
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
