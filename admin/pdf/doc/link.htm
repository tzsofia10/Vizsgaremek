Link

Link(float x, float y, float w, float h, mixed link)

Leírás

Beszúr egy hivatkozást egy téglalap alakú területére az oldalnak. Szöveg és kép hivatkozások beillesztése általában a Cell(), Write() vagy Image()-en keresztül történnek, de ez a módszer hasznos lehet például egy klikkelhető terület létrehozásában egy képen belül.

Paraméterek

x
A téglalap bal felső sarkának x koordinátája.
y
A téglalap bal felső sarkának y koordinátája.
w
A téglalap szélessége.
h
A téglalap magassága.
link
URL vagy azonosító, amellyel a AddLink() visszatér.

Lásd még

AddLink, Cell, Write, Image
