Image

Image(string file [, float x [, float y [, float w [, float h [, string type [, mixed link]]]]]])

Leírás

Beszúr egy képet. A megjelenítendő mérete különböző módokon adható meg:
  • meghatározott szélesség és magasság (a felhasználó mértékegységében kifejezve vagy dpi-ben)
  • egyik érték megadásával, ilyenkor a másik érték automatikus meghatározásra kerül és az eredeti képarány megmarad
  • nincs megadva egyik érték sem, ebben az esetben a kép 96 dpi felbontásban jelenik meg
Támogatott formátumok: JPEG, PNG és GIF. A GD kiterjesztés szükséges a GIF-hez.

A JPEG-nek az összes sajátossága elérhető:
  • szürkeárnyalatos
  • true colors (24 bit)
  • CMYK (32 bit)
PNG-nél engedélyezett:
  • szürkeárnyalatos legfeljebb 8 bit
  • indexed colors
  • true colors (24 bit)
Gifeknél, amennyiben animált gif, csak az első képkocka jelenik meg.

Átlátszóság (transzparencia) támogatva.

A formátum megadható egyénileg vagy meghatározásra kerül a kiterjesztésből.

Hivatkozás megadása a képre szintén lehetséges.

Megjegyzés: ha egy kép többször is be van illesztve, csak egy másolata lesz beleágyazva a dokumentumba.

Paraméterek

file
Elérési út vagy URL a képhez.
x
A bal felső sarok x koordinátája. Ha nincs megadva vagy értéke null -al egyenlő, az aktuális x pozíció kerül használatra.
y
A bal felső sarok y koordinátája. Ha nincs megadva vagy értéke null -al egyenlő, az aktuális y pozíció kerül használatra. Amennyiben szükséges oldaltörés, akkor bekövetkezik (illetve csak akkor ha az automatikus oldaltörés engedélyezve van), majd a függvény hívás után az aktuális y koordináta a kép aljára kerül át.
w
A kép szélessége az oldalon. Három eset lehetséges:
  • Ha az érték pozitív, megadja a szélességet a felhasználó egységében.
  • Ha az érték negatív, a tényleges érték a vízszintes felbontásnak megfelelően, dpi-ben kerül felhasználásra.
  • Ha nem beállított vagy nullával egyenlő, automatikusan kerül kiszámításra
h
A kép magassága az oldalon. Három eset lehetséges:
  • Ha az érték pozitív, megadja a magasságot a felhasználó egységében.
  • Ha az érték negatív, a tényleges érték a függőleges felbontásnak megfelelően, dpi-ben kerül felhasználásra.
  • Ha nem beállított vagy nullával egyenlő, automatikusan kerül kiszámításra
type
Kép formátum. Lehetséges értékek (nem betűméret érzékeny): JPG, JPEG, PNG és GIF. Ha nincs megadva, a kiterjesztésből kerül meghatározásra.
link
URL vagy azonosító, amivel az AddLink() tér vissza.

Példa

// Beilleszt egy logót a bal felső saroktól 300 dpi-re (vízszintesen)
$pdf->Image('logo.png',10,10,-300);
// Beilleszt egy dinamikus képet egy URL-ből
$pdf->Image('http://chart.googleapis.com/chart?cht=p3&chd=t:60,40&chs=250x100&chl=Hello|World',60,30,90,0,'PNG');
