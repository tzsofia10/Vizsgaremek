Text

Text(float x, float y, string txt)

Leírás

Megjelenít egy stringet. Az első karakter bal oldala a viszonyítási alap. Ez a függvény lehetőséget ad egy string tetszőleges elhelyezésére az oldalon, de általában string kiíratásra a következő függvények használatosak: Cell(), MultiCell() vagy Write().

Paraméterek

x
Az x koordináta értéke.
y
Az y koordináta értéke.
txt
Kiírandó sztring.

Lásd még

SetFont, SetTextColor, Cell, MultiCell, Write
