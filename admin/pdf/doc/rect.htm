Rect

Rect(float x, float y, float w, float h [, string style])

Leírás

Megjelenít egy téglalapot. Lehet csak kerettel, kitöltött, vagy mindkettő.

Paraméterek

x
A bal felső sarok x koordinátája.
y
A bal felső sarok y koordinátája.
w
Szélessége.
h
Magassága.
style
Megjelenítési stílus. Lehetséges értékek:
  • D vagy üres string: keretes. Ez az alapértelmezett.
  • F: kitöltött
  • DF vagy FD: mindkettő

Lásd még

SetLineWidth, SetDrawColor, SetFillColor
