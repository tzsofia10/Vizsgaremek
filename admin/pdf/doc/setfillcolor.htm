SetFillColor

SetFillColor(int r [, int g, int b])

Leírás

Beállít egy színt az összes kitöltési műveletre (téglalapok és cellák kitöltési illetve háttér színe). Kifejezhető RGB komponensekkel vagy szürkeányalatosan. A művelet meghívható az első oldal létrehozása előtt és értéke megmarad oldalról oldalra.

Paraméterek

r
Ha g és b megadott, piros komponens; Ha nem, szürkeárnyalat. Értéke 0 - 255 között.
g
Zöld komponens (érték 0 - 255 között).
b
Kék komponens (érték 0 - 255 között).

Lásd még

SetDrawColor, SetTextColor, Rect, Cell, MultiCell
