SetLink

SetLink(int link [, float y [, int page]])

Leírás

Beszúr egy belsőhivatkozást. Beállítja az oldalt és a pozíciót, ahová a hivatkozás mutat.

Paraméterek

link
A hivatkozás azonosítója amit az AddLink() ad vissza.
y
A végcél y koordinátája. A -1 az aktuális pozíció.
Alapértéke: 0 (a lap teteje).
page
A céloldal oldalszáma. A -1 az aktuális oldal és ez is az alapérték.
