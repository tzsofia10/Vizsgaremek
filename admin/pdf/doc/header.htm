Header

Header()

Leírás

Ez a függvény felel az oldal fejlécének beillesztéséért. Automatikusan meghívásra kerül az AddPage() és a Close() által, így nem érdemes közvetlenül meghívni. Alapértelmezettként az FPDF-ben üres, így be kell állítani, ha fejlécet szeretnénk.

Példa

class PDF extends FPDF
{
function Header()
{
    // Arial, félkövér, 15 pt-os betűtípus kiválasztása
    $this->SetFont('Arial','B',15);
    // Jobbra igazítás
    $this->Cell(80);
    // Bekeretezett cím
    $this->Cell(30,10,'Title',1,0,'C');
    // Sortörés
    $this->Ln(20);
}
}
